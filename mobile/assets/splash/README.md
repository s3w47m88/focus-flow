## Splash Screen Guidelines

1. Keep it simple - usually just logo on solid background
2. Match your app's dark theme (#000000 background)
3. Center your logo/brand mark
4. Don't include text that might need localization
5. Avoid UI elements that might change

## Design Tips

- Use same background color as your app's initial screen
- Keep file sizes optimized
- Test on actual devices when possible
- Consider using a gradient or subtle pattern
