# App Icons

This directory should contain app icons in the following sizes for iOS:

## Required Icon Sizes

- `icon-20.png` - 20x20
- `icon-20@2x.png` - 40x40
- `icon-20@3x.png` - 60x60
- `icon-29.png` - 29x29
- `icon-29@2x.png` - 58x58
- `icon-29@3x.png` - 87x87
- `icon-40.png` - 40x40
- `icon-40@2x.png` - 80x80
- `icon-40@3x.png` - 120x120
- `icon-60@2x.png` - 120x120
- `icon-60@3x.png` - 180x180
- `icon-76.png` - 76x76
- `icon-76@2x.png` - 152x152
- `icon-83.5@2x.png` - 167x167
- `icon-1024.png` - 1024x1024 (App Store)

## Icon Guidelines

1. Use a 1024x1024 master icon
2. No transparency (iOS adds rounded corners automatically)
3. Simple, recognizable design
4. Test at small sizes for clarity

## Generating Icons

You can use tools like:
- Capacitor's `@capacitor/assets` package
- Online generators like makeappicon.com
- Design tools like Figma or Sketch with export presets
